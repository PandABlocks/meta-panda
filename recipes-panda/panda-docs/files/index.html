PandA

This page lists documentation for the components of this PandA

Rootfs

The rootfs is responsible for booting the version of Linux that powers this device, including setting up networking.

Rootfs FAQ

FPGA

The FPGA provides the implementation of the logical Blocks that are wired together for a given application. It exposes a register interface to the TCP server so that the parameters and wiring of these Blocks can be changed at runtime.

FPGA Documentation

TCP Server

The TCP server is meant for integration into Control systems like EPICS and Tango. It exposes an ASCII Control interface on port 8888 and a binary Data interface on port 8889.

TCP Server Documentation

Web Control

Web Control provides a webserver that exposes a browser based Graphical User Interface that can be used to inspect, configure and modify the system.

Web Control Documentation
